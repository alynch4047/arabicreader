
                    Baghdad - Type 1 Postscript
                    Arabic (Persian, Urdu) font
                        (for Windows 3 with
                        Adobe Type Manager)



   This is just a font, and does not include keyboard mapping software
(characters above 127 can be entered with the Alt key and numeric keypad,
or through the Character Map accesory), nor right-to-left text entry support
(there are several small shareware Windows utilities for this purpose).
Install this Type 1 font the usual way (through the ATM control panel).

History:

v1.0:  Initial release.
v2.0:  Cleaned up inconsistencies; added
       Baghdad88596 and Baghdad449 fonts.

Character Code Mapping:

   For maximum flexibility, three fonts are included in this package,
differing only in their character code mapping (font encoding).  The
plain Baghdad font (the BAGHD___.* files) uses a pragmatic non-standard
encoding, which tries to put the most important characters in the range
32-126.  The Baghdad88596 font (the BGHD6___.* files) uses an encoding
which is compatible with the ISO 8859-6 standard (Latin 6) in the range
160-242; while the Baghdad449 font (the BGD49___.* files) uses an
encoding which is compatible with ASMO-449 in the range 32-116.

Pragmatic Encoding:

This font contains Arabic glyphs, rather than abstract characters, so
that positionally-contextual forms are encoded separately.  The Arabic
glyphs are assigned character codes so that isolated forms are in the
character code positions where the Roman upper case characters are in the
ASCII standard, and doubly-connected forms (or the final forms of those
letters which do not connect to the left) are in the ASCII Roman lower
case character positions.  As far as possible, these glyphs correspond to
letters in ASCII which have comparable sound-values (for example, the
letter _lam_ corresponds to ASCII L/l).  Generally, initial forms are on
upper case character positions + 128, and final forms are on lower case
characters + 128.  The vowel diacritics, some miscellaneous characters,
and Persian and Urdu letters are in positions 130-191.  The following
table equates this font's glyphs with Unicode (ISO 10646 basic plane)
